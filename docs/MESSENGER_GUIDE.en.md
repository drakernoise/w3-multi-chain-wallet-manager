### "Username already taken"
If the name you chose is already in use, please try another one. Usernames are currently permanent.

### Connection Failed
If you see "Connection Failed", the chat server might be down or your connection blocked. Click **Retry Connection** to try again.

### Stalled Identity
If you have trouble logging in and the server says you are already logged in or there's a registration error, use the **"Clear Stalled Identity"** button on the error screen. This will reset your local chat session.
